Color memory game for iPhone

See the color. Remember the tone.

Chroma Echo is a modern memory game where you watch a color, rebuild the closest shade from a precise palette, and push your visual memory through Color Quest, sequence challenges, shape memory, and Game Center rankings.

Built for color memory, not simple guessing.

Chroma Echo uses a palette-based matching system, so every round tests how closely you remember the exact tone.

Color Quest

Map progression

Move through staged challenges, unlock new levels, and face stronger memory trials as you progress.

Sequence

Multiple colors

Some stages ask you to remember two or three colors in order, then match each one from the palette.

Shapes

Shape + color memory
